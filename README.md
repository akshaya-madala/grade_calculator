 Student Grade Calculator

* Project Description
The Student Grade Calculator is a simple Python program that accepts a student's name and marks, calculates the grade using if-elif-else statements, and displays an encouraging message based on the grade.

*Features
- Accepts student name and marks.
- Validates marks (0–100).
- Uses a while loop for invalid inputs.
- Uses a function to calculate the grade.
- Displays encouraging messages.

* Grading Criteria

| Marks | Grade |
|-------|-------|
| 90-100 | A |
| 80-89 | B |
| 70-79 | C |
| 60-69 | D |
| 0-59 | F |

* Technologies Used
- Python 3
- Jupyter Notebook (Development)

* How to Run

bash
python grade_calculator.py
